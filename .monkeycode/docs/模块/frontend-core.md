# 前端核心模块
frontend/src/core/ 目录包含前端的核心业务逻辑，包括 API 客户端、线程管理、数据模型等。

## 结构
```
frontend/src/core/
├── api/                  # API 客户端
├── threads/              # 线程管理
├── models/               # 数据模型
├── skills/               # 技能系统
├── mcp/                  # MCP 集成
├── messages/            # 消息处理
├── settings/             # 用户设置
├── todos/                # Todo 系统
├── config/               # 应用配置
├── i18n/                 # 国际化
├── utils/                # 工具函数
└── artifacts/            # 制品管理
```

## 核心模块
- **api/**: 与后端的 API 通信
- **threads/**: 管理对话线程
- **models/**: 数据类型和模型
- **skills/**: 技能展示和管理
- **mcp/**: MCP 服务器集成
- **messages/**: 消息处理和渲染
- **settings/**: 用户设置和偏好
- **todos/**: 任务列表管理
